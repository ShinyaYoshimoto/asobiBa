# @asobiba/mahjong-api

This package contains the mahjong-related API implementations.

## Contents

- `hands`: Hand information API
- `score-declarations`: Score declaration APIs (get, post, calculate, summaries)
- `scores`: Score answer API

## Modules

- Hand domain and infrastructure
- Score domain and infrastructure
- Answer domain and infrastructure

## Usage

```typescript
import { HandsGetHander, ScoreDeclarationsGetHandler } from '@asobiba/mahjong-api';
```

## Note

This is a copy of the implementation from the `mahjong-hono` directory for monorepo preparation.
