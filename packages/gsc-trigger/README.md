# @asobiba/gsc-trigger

This package contains the Google Cloud Storage trigger implementation for handling file uploads.

## Contents

- `trigger-upload-gcs`: Handler for GCS upload events (ZIP and image files)

## Usage

```typescript
import { TriggerUploadGcs } from '@asobiba/gsc-trigger';
```

## Note

This is a copy of the implementation from the `triggers` directory for monorepo preparation.
