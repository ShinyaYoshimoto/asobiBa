# @asobiba/share-photo-api

This package contains the photo sharing API implementations.

## Contents

- `photos`: Photo management APIs (post, search, upload-signed-url, description, tags)
- `tags`: Tag management API

## Modules

- Photo domain and infrastructure
- Tag domain and infrastructure

## Usage

```typescript
import { PhotosSearchPostHandler, TagsGetHandler } from '@asobiba/share-photo-api';
```

## Note

This is a copy of the implementation from the `mahjong-hono` directory for monorepo preparation.
